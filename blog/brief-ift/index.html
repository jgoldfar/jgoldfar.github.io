Motivating Example

Consider a curve such as that defining a unit circle in the plane, \[x^{2} + y^{2}=1\] Note that this definition is a special case of a more general situation, a curve defined as a level set of a function \[f(x,y):= x^{2} + y^{2}= 1 =:z_{0}\] Since this is a relatively simple curve, we can draw it and observe that, while it is not a function with respect to either \(x\) or \(y\) (that is, in general we cannot solve the equation for either \(y\) or \(x\), respectively), in a small neighborhood of any point (signified by a dashed circle in Figure 1 about the given point) the curve is indeed a function, as we can see:

When is it not possible to solve this equation locally for, say, \(y\)? Evidently, the issue arises in any neighborhood of \((1,0)\) or \((-1,0)\), where the slope of the tangent line with respect to the other variable is undefined. This is a bit hard to quantify and/or generalize to more than 2 dimensions, but it is equivalent to check that \(\frac{\partial{} f}{\partial{} y} \neq 0\).

Now, most of the time, we have no idea how an implicit function looks: how does the graph of \[\tan(y) - y = x + e^{x}\] look? Where can it be solved for \(y\) in some sense? Evidently, we won’t be able to write down a representation \(y=f(x)\) in an explicit closed form, but the notion of a closed form solution itself depends on the set of functions one considers to be elementary, so this isn’t a particularly useful criterion anyways. The existence of such a representation, even if we can’t find it in closed form, is still useful: indeed, this is exactly the trick used in the implicit differentiation technique commonly used in freshman-level Calculus courses.

Note that the previous equation can be written in the implicit form \[F(x,y) := \tan(y) - y - x - e^x \equiv 0\] and if an explicit representation \(y=f(x)\) existed, it must satisfy \(F(x,f(x))=0\). Implicit function theorem offers a way to prove that such a representation exists under certain conditions.

Statement of Implicit Function Theorem

Implicit function theorem can be found in many references; we will use that from (Evans 1998, vol. 19, sec. C.5.7). Fix any \(n,m \in \mathbb{Z}_{+}\). We will consider points in \(\mathbb{R}^{n+m}\) as pairs of points in the Cartesian product \[\mathbb{R}^{n}\times \mathbb{R}^{m} = \left\{(x,y):~x=(x_{1},\ldots,x_{n}) \in \mathbb{R}^{n},~y=(y_{1},\ldots,y_{m})\right\}\]

Let \(U\subset \R^{n+m}\) be open and \(f:U \to \R^{m}\) be a \(C^{1}\) map. That is, \(f \in C^{1}(U; \R^{m})\). Denote the components of \(f(x,y) \in \R^{m}\) as \(f^{1},\ldots,f^{m}\). Fix any \(\big(x_{0}, y_{0}\big) \in U\) and set \(z_{0} = f(x_{0}, y_{0})\). Suppose that \[\text{det}(\nabla_{y} f) = \begin{vmatrix} f_{y_{1}}^{1} & \cdots & f_{y_{m}}^{1} \\ & \ddots & \\ f_{y_{1}}^{m} & \cdots & f_{y_{m}}^{m} \end{vmatrix} \neq 0\] Then there is an open set \(V\) with \(U \supset V \ni (x_{0}, y_{0})\), an open set \(W\) with \(\R^{n} \supset W \ni x_{0}\) and a function \(g \in C^{1}(W;\R^{m})\) such that \[g(x_{0}) = y_{0}~ \text{and}~ f(x, g(x)) = z_{0}~\text{for}~x \in W\] \(g\) is unique, in that if \((x,y) \in V\) and \(f(x,y) = z_{0}\), then \(y=g(x)\). Moreover, increasing the regularity of \(f\) increases the regularity of \(g\).

Application and Extension

Consider the implicit form for the equation defining a unit circle, \[F(x,y) := x^2 + y^2 - 1 \equiv 0\]

We calculate \[\nabla_y F = F_y = 2y\] and hence \(\text{det} (\nabla_y F) = 0\) for \(y \neq 0\); that is, the implicit function has an explicit form for \(y \neq 0\). Revisiting Figure 1, we see that this makes perfect sense: anywhere \(y=0\), the graph is locally vertical. The set \(V\) of the implicit function theorem can be identified with, for example, the interior of the dashed circle in Figure 1, and its projection to the \(x\)-axis with the set \(W\) of the implicit function theorem.

In freshman-level Calculus courses, we typically use implicit function theorem to give a justification for implicit differentiation; recall that we can use chain rule as a kind of change of variables rule: \[\dD{f}{x} = \dD{f}{y}\dD{y}{x}\] which we use to differentiate a (perhaps complicated) implicit function of the form \(F(x,y)=0\). From one side, we could use multivariable Calculus to complete this task, since it follows that \[0 = \D{}{x}F(x,y) = F_x + F_y y'\] so \[y' = \frac{-F_x}{F_y}\] Note that this is exactly why we don’t encourage students to think of differentials as quotients, since the above formula would read \[\dD{y}{x} = - \D{F}{x} \Big/ \D{F}{y}\] That is, cancellation does not work with all differentials, so learning it as a rule would be a mistake.

If multivariable Calculus is not available, we actually have to use implicit function theorem “behind the scenes”, even though the prerequisites for the result include multivariable calculus! Given an implicit function \(F(x,y) = 0\), we know that we can “solve” this equation for \(y\) in a neighborhood \(V\) of a solution \((x_0,y_0)\) to get a differentiable representation \(y=g(x)\) such that \(F(x,g(x))=0\) for \((x,y) \in V\). Therefore \[y' = g'(x)\] exists; by uniqueness of the derivative, the two derivative functions must be identical. The important thing here is that the representation is differentiable, which justifies seeking a derivative in the first place!

Sketch of the Proof

Implicit function theorem actually follows from another, more “powerful” result, the inverse function theorem.

Let \(A \subset \R^n\) be open and \(f:A \to \R^n\) be continuously differentiable with Jacobian \(Df\). Let \(x_0 \in A\) satisfy \(Jf(x_0) := \text{det}(Df(x_0)) \neq 0\). Then there exists a neighborhood \(U \ni x_0\) with \(U \subset A\) and an open neighborhood \(W\) of \(f(x_0)\) such that \(f(U)=W\) and the restriction of \(f\) to \(U\) has a continuously differentiable inverse \(f^{-1}:W \to U\). Moreover, for any \(y \in W\) and corresponding \(x=f^{-1}(y)\), the Jacobian of the inverse \(Df^{-1}(y)\) is given by \[Df^{-1}(y) = [Df(x)]^{-1}\]

Inverse function theorem is actually true in quite general spaces The idea of the proof of the implicit function theorem is to consider a function \(F(x,y) = (x,f(x,y))\); if we want to find \(y=g(x)\) such that \(f(x,g(x))=0\), it is enough to show that \(F^{-1}\) exists near some solution of \(f(x,y)=0\).

Future Work

Implicit function theorem can be generalized quite a bit. This will be the topic of a subsequent post if it will be useful. The same could be said of the inverse function theorem.

Plotting packages for implicit equations are based on the implicit function theorem along with other facts and heuristics; it would be interesting to see how far an implicit plotting package could get with just implicit function theorem, in some sense.
